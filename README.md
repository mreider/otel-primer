# OpenTelemetry Primer

A single-page, progressive primer on OpenTelemetry concepts. Each entry builds on the last, designed around constructivist learning principles.

Live at [otel-primer.mreider.com](https://otel-primer.mreider.com)

## Structure

- `index.html` — the entire primer, one page
- `style.css` — print-inspired typography using Crimson Text
- `CNAME` — custom domain for GitHub Pages
